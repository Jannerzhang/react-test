# 项目名称

## 简介
本项目是一个基于 React 的前端应用，使用 TypeScript 进行开发，集成了 TDesign 组件库和 Emotion 样式库，提供了丰富的用户界面和交互功能。

## 功能
- 用户信息展示
- 策略记录管理
- 投诉记录查看
- 交易信息管理

## 技术栈
- React
- TypeScript
- TDesign
- Emotion

## 安装与运行
1. 安装依赖：
   ```bash
   yarn install
   ```
2. 启动项目：
   ```bash
   yarn start
   ```

## 目录结构
```
public/
  index.html
  manifest.json
src/
  App.tsx
  index.tsx
  components/
  styles/
```
